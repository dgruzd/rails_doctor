# RailsDoctor

Gem is used to find troubles in rails code, e.g. associations without index

## Installation

Add this line to your application's Gemfile:

    gem 'rails_doctor'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rails_doctor

## Usage

    doctor = RailsDoctor.new
    doctor.analize
